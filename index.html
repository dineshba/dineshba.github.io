Exec

exec (buildin command) We can use exec for two use cases (as far as I learned today :-P) To create independent shell from current shell Usual way: $ bash # created a bash shell $ #do whatever you wanted $ exit # will go back to the shell which it created Using exec: $ exec bash #created a bash shell and killed the current shell $ #do whatever you wanted $ exit # will not go back to the shell which it created.
